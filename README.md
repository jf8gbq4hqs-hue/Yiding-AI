## 产品介绍
懿鼎AI是一款注重隐私安全、本地存储的多功能AI助手APP，核心特性如下：
- 支持智能聊天、代码处理、文生图、图生图、多窗口管理等功能
- 所有用户数据优先本地保存，不上传云端，安全可控
